# Yoisaki Kanade-Skill (宵崎奏.Skill)

#### 正如简介所说，这是一个基于ProjectSekai(世界计划)游戏中的一名角色“宵崎奏”(Yoisaki Kanade)所制作的Skill

#### 首先需要明确的是，这是一个主要用于AI聊天的skill，没有额外技能

---

### 使用教程
- `Initial-Markdown`文件夹下的文件为制作Skill所用到的原始Markdown文件/文档，文档包含超过8000字的完整设定，涵盖`角色心理学`、`语言模型`、`对话示例`、`技术配置`等全方位内容。包含超长且完整的细节，`角色设定`、`对话风格`、`知识库`、`互动模式`等 (理论上也可以直接当做Skill使用)
- `Kanade-skill`文件夹则为Skill的成品。下载完成main分支的压缩包后，需将压缩包内的`Kanade-skill`文件夹解压缩至一个任意记住的位置。之后将解压出来的整个`Kanade-skill`文件夹拖入任意支持Skill的AI即可使用 (建议在任意盘符根目录下新建一个文件夹，用于存放解压缩的内容，避免乱放影响其他软件或系统)

---

### Skill制作所使用的程序及开源项目
- `Initial-Markdown`文件夹中的Markdown文件是由Kimi (`www.kimi.com`) 提供的数据收集及整理 (目前仅`Yoisaki_Kanade-Skill-202607202230.md`是由Kimi处理的)
- `Kanade-skill`文件夹下的Skill成品则为开源项目`https://github.com/JodieRuth/GalgameCharacterSkills`提供的便携处理。主要用于将`Initial-Markdown`文件夹下的Markdown文件通过API调用AI模型，将其内容生成为SKill成品
- Skill成品是由在线AI模型`deepseek-v4-pro`所生成，其API调用程序已在前面的内容中提到
